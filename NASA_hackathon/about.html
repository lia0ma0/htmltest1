Download for Windows

An easy guide to set up Etones for Windows

Steps to guide you!

1. Click download for Windows in top right of the website

2. Click Download ETONES

3. Click to where the ETONES file was downloaded in your computer

4. Unzip the folder by right click on it, press 7-zip -> Extract File

5. Click on “ETONES” to run the Application

6. After you see two slots for you to input, go back to the website to copy a pair of images from the image page. It has to be the same region, but doesn't need to be at the same time(ex. Fuji Mountain from 2013 and Fuji Mountain from 2021)

7. Input the file name, composer, race, and image processor

8. Click on export to download the music created by the two images, the music file will be located in the download file after it’s done.

9. After downloading the music file, click Download Musescore

10.Download Musescore for Windows or “Free Download” if you are in the home page

11. Click to where Musescore file was downloaded in your computer(Normally it should be in your download folder)

12. Run the installation exe; Click next until you see install in the screen(Remember to click on the agree the terms in License Button in the second page during the installation to keep on going)

13. Run Musescore

14. Click on the FIle on the top right, then click open. Click on the song that is downloaded through ETONES.

15. Run the song by clicking on the play button in the middle top of Musescore
